OVBS Studio
===========

OVBS Studio is a Linux-first, vibe-driven fork of OBS that builds on the
project's capture, compositing, encoding, recording, and streaming foundation
while moving in its own direction.

This repository is not a clone of upstream policy. We start from what we got
and keep evolving toward a better, more flexible OBS-style experience with
local tweaks, AI-friendly workflows, and developer-first tooling.

Contributing
------------

- If you want to contribute code, start with this repository's
  `CONTRIBUTING.md`.
- Follow the repo's current style configuration and use the tools provided
  here when available.
- This project is driven by maintainers and contributors working together.
  Pull requests are welcome, and maintainers decide whether a change fits the
  current direction.
